算法与数据结构系列-二分查找

发表于 2020-05-24 | 分类于 算法与数据结构

1. 概念

二分查找也称折半查找（Binary Search），它是一种效率较高的查找方法。折半查找要求线性表必须采用顺序存储结构，而且表中元素按关键字有序排列。

最常用的二分查找场景有以下几种：

  • 查找给定值
  • 查找给定值的左侧边界
  • 查找给定值的右侧边界

2. 二分查找的技巧

  1. 计算mid时防止溢出，建议写成：int mid = left + (right - left) / 2;
  2. 查找时不要使用 else 分支，而是把所有情况用 else if 写清楚，展现所有细节。
  3. while 循环条件关联搜索区间：当搜索区间是两端闭区间 [left, right]时，while 条件是 left <= right；当搜索区间是左闭右开区间 [left, right)时，因为索引大小 nums.length 是越界的，while 条件是 left < right。
  4. 每次更新查找边界时，需要保证当前 mid 在下一次查找的搜索范围之外。
