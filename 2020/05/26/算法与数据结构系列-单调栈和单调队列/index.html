算法与数据结构系列-单调栈和单调队列

发表于 2020-05-26 | 分类于 算法与数据结构

1. 概念

1.1 单调栈

栈是一种后进先出（LIFO）的数组结构，单调栈在保证满足栈的基本特性的前提下，从栈顶到栈底的元素是严格递增或递减的。

注意，递增和递减的方向是指：从栈顶到栈底。

具体的入栈和出栈过程如下：

  • 对于单调递增栈，若当前入栈元素为e，从栈顶开始遍历元素，把小于或等于e的元素弹出栈，直接遇到一个大于e的元素或者栈为空为止，然后再把e压入栈中。
  • 对于单调递减栈，则每次弹出的是大于或者等于e的元素。

1.2 单调队列

队列是一种先进先出（FIFO）的数据结构，单调队列在保证满足队列的基本特性的前提下，从队头到队尾的元素是严格递增或递减的。

注意，递增和递减的方向是指：从队首到队尾。

具体的入队和出队过程如下：

  • 对于单调递增队列，若当前入队的元素为e，从队尾开始开始遍历元素，把大于或相等e的元素出队，直到遇到一个小于e的元素或者队列为空为止，然后把当前元素e插入到队尾。
  • 对于单调递减队列，则每次从队尾删除的是小于或相等e的元素。

2. 单调栈和单调队列的异同点

  • 单调栈和单调队列都具备单调性。
  • 单调栈和单调队列维护的时间复杂度都是O(n)，因为每个元素都只操作一次。
  • 单调栈和单调队列维护的区间不同，当访问到第i个元素时，单调栈维护的区间为[0, i)，而单调队列维护的区间为(lastpop, i)。
  • 单调队列可以从两端访问（队首和队尾），而单调栈只能从一端访问（栈顶），这也导致单调栈无法获取[0, i)的区间最大值/最小值。

3. 单调栈和单调队列的应用

3.1 单调栈

  • 查找元素e左边区间第一个比它小的数，第一个比它大的数
  • 确定元素e是否是区间最值
  • 查找元素e右边区间第一个大于它的值，以及两者之间的距离
  • 确定以元素e为最值的最长子区间

3.2 单调队列

  • 查询区间最值（不能维护区间k大，因为队列中很有可能没有k个元素）。
  • 优化DP，单调队列可作为一种特殊数据结构用于优化动态规划方面问题，且多数情况是与定长连续子区间问题相关联。
